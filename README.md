### Instrucciones

> - Para ejecutar en ambiente de desarrollo

```
    npm run dev
```

> - Para generar fuentes producción [dist](./dist/)

```
    npm run build
```

> - Para ejecutar fuentes de producción [dist](./dist/)

```
    npm run start
```

> - Para ejecutar pruebas unitarias

```
    npm run test
```
